# Unified_Customer_comms
Creating an AI Agent to unify and automate customer communications and scheduling for a small business.

# scope
Initial scope is centered aroung Meta messaging capabilities (Facebook messenger, Instagram DM, and WhatsApp) and creating an AI Agent to interface with customer requests for business, assist in parsing out quote information and creating an event in google calendar.

# backlog
- Expanding to text/cell service requests
- Integrating with Jobber CRM to track completeness
- Requesting google reviews after services rendered.
